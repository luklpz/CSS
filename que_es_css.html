Ejercicio: El CSS.

Primero responderemos al interrogante más básico: ¿Qué es CSS?

El CSS (Cascading Style Sheets) funciona para poder crear páginas web visualmente más atractivas mediante funciones como definir el color de los textos, modificar el tamaño de los encabezados, crear disposiciones de columnas, la adición de barras laterales...

Una vez sabido lo anterior, nos podemos preguntar: ¿Qué formas hay de aplicar el CSS?

El CSS lo podemos aplicar de tres formas distintas:

- CSS interno - Este estilo consiste en incrustar la etiqueta <stile> en la sección <head> del documento. Una vez hecho eso, podemos añadir las propiedades CSS que queramos dotar a los elementos del documento.

- CSS externo - Mediante este método haremos uso del CSS mediante un archivo externo a la página el cual enlazaremos con la misma, pudiendo cambiar toda la página web únicamente modificando dicho archivo. Para llevar a cabo este método, primero crearémos un archivo .css en el que añadiremos las reglas que deseemos y a continuación enlazaremos la página con dicho archivo mediante el siguiente comando: <link rel="stylesheet" type="text/css" href="style.css" />.

- CSS Inline - Mediante este último método se aplicará el estilo mediante la integración del CSS en la misma etiqueta <> de la cual queremos modificar.
